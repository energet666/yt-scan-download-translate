Сервис сканирования и загрузки из плейлистов YouTube с автоматическим переводом.

## Основные возможности
*   🔄 **Автоматический мониторинг**: периодическая проверка плейлистов на наличие новых видео.
*   📥 **Умная загрузка**: скачивание только новых видео (проверка по ID).
*   🎙️ **Голосовой перевод**: автоматическое наложение аудиодорожки (через VOT) с помощью `ffmpeg`.
*   🔁 **Retry + TTS Fallback**: до 3 попыток с живыми голосами, при неудаче — автопереключение на TTS.
*   🔇 **Пропуск видео без речи**: автоматическое определение и пропуск видео, в которых нет речи.
*   🧹 **Авто-очистка**: удаление временных файлов (`.mp3` и исходное видео) после успешной сборки.
*   🚀 **Интерактивная настройка**: мастер первого запуска для быстрой конфигурации в консоли.
*   🛡️ **Надежность**: корректное завершение работы (Graceful Shutdown) и ведение структурированных логов.

## Требования
Для работы сервиса должны быть установлены:
*   [go](https://go.dev) — для компиляции
*   [yt-dlp](https://github.com/yt-dlp/yt-dlp) — для работы с YouTube
*   [ffmpeg](https://ffmpeg.org) — для сборки видео контейнера
*   [vot-cli-live](https://github.com/fantomcheg/vot-cli-live) — для загрузки голосового перевода

## Быстрый старт
1.  Скомпилируйте или запустите проект:
    ```bash
    go run .
    ```
2.  При **первом запуске** сервис автоматически:
    *   Создаст папку `.private/` с необходимыми шаблонами.
    *   Запустит **интерактивный мастер настройки** в терминале.
3.  В мастере вы сможете:
    *   Указать URL одного или нескольких плейлистов.
    *   Выбрать, нужен ли автоматический перевод для каждого плейлиста.
    *   Указать, нужно ли скачивать существующие видео или только те, что появятся в будущем.

## Конфигурация
После настройки все параметры сохраняются в `.private/config.json`. Вы можете редактировать их вручную позже:
```json
[
  {
    "playlistURL": "https://...",
    "translate": true
  }
]
```

## Управление
*   Сервис проверяет плейлисты каждую минуту.
*   Для остановки используйте `Ctrl+C`. Сервис безопасно завершит текущую операцию и сохранит состояние.
